# react-planets

## Install

```bash
npm install --save react-planets
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'react-planets'
import 'react-planets/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
